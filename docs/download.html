Data set

The data set can be downloaded in different versions. The .json and .tsv (tab-separated values) versions contain the full data set. Scripts illustrating importing the data into Python and R can be accessed below.

  • manynames.json
  • manynames.tsv

Images

The ManyNames images were sampled from the Visual Genome (VG) images. The original VG images can be downloaded at visualgenome.org. These images were rescaled and displayed with a red bounding box for the data collection. This version of the image can be downloaded at object-naming-amore.upf.edu.

Scripts

He we provide scripts to faciliate accessing the dataset in Python and R. In addition there are scripts to a specific download images using the .csv generated in the search interface and scripts to recreate the ManyNames images from the VisualGenome versions.
